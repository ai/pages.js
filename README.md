# Pages.js

## Contributing

1. To run tests you need node.js and npm. For example, in Ubuntu run

   ```
   sudo apt-get install nodejs npm
   ```

2. Next install npm dependencies:

   ```
   npm install
   ```

3. Run test server:

   ```
   ./node_modules/.bin/cake test
   ```

4. And open tests in browser: <localhost:8000>.
